## 使用教程

（以 edge 界面为准）

1. 在浏览器自带的扩展商店下载Tampermonkey（篡改猴）(直接搜索即可)
或者打开这个网址：[篡改猴](https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/iikmkjmpaadaobahmlepeloendndfphd)

2. 启用Tampermonkey，在“扩展”（像一块拼图）下拉菜单中选择“Tampermonkey”，点击“管理面板”

3. 从文件夹中把“fls惩罚者-1.0.user.js”拖进浏览器窗口，然后点“安装”，即可自动安装脚本

4. 点击浏览器窗口上方的“扩展”图标，然后点“管理扩展”，打开左下角的“开发人员模式”

5. 点击“Tampermonkey”下面的“详细信息”，在打开的页面中勾选“允许用户脚本”

*现在你可以自由切屏了*
